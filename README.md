# Practical-Statistics-for-Data-Scientist
Tóm tắt sách và các đoạn code đi kèm 
Nội dung chính của sách bao gồm các phần sau    
Mỗi chương bao gồm nhiều bài viết nhỏ dạng text với một file code được đính kèm theo

#### Chương 1: EDA dữ liệu 
- Một bản tóm tắt về các câu hỏi đặt ra với dữ liệu, một số biểu đồ trực quan hóa các loại dữ liệu
- 2 bản code để EDA trên bộ dữ liệu cụ thể 

#### Chương 2: Các phân phối trong dữ liệu 
- Các vấn đề dữ liệu và phân phối của chúng: các phân phối phổ biến (có code đi kèm)
- Khái niệm về thống kê mẫu và ước lượng cho thống kê mẫu (có code đi kèm)
- Sử dụng phương pháp bootstrap để ước lượng cho thống kê mẫu (có code đi kèm)
- Vấn đề về bias trong dữ liệu (có code đi kèm) 

#### Chương 3: Kiểm định
- Các khái niệm cơ bản trong kiểm định thống kê (có code đi kèm)
- Các loại test cơ bản (có code đi kèm)

#### Chương 4: Hồi quy 
- 2 case study (bao gồm code và phân tích) đặc trưng cho 2 bài toán là fixed và random X 
- Mỗi case study bao gồm (mô hình hóa bài toán, giả định với dữ liệu, công thức ước lượng tham số, cách kiểm định các giả thuyết trên một bộ dữ liệu, thuật toán tính toán thực tế, các hiện tượng có thể xảy ra, phương pháp phân tích kết quả -> quy trình cuối cùng để thực hiện một bài toán hồi quy thế này)
  
#### Chương 5: Phân loại
- 3 phương pháp khác nhau cho bài toán phân loại (quy trình trình bày giống y như ở trên)

- Các vấn đề với dữ liệu 

#### Chương 6: Các thuật toán học máy giải bài toán hồi quy và phân loại 
- Trình bày về cách hoạt động và code của từng thuật toán, các vấn đề của từng thuật toán và xử lý thế nào.
  
#### Chương 7: Học không giám sát (PCA và bài toán phân cụm) 
- Tổng quan 2 bài toán PCA và phân cụm
- Thuật toán PCA
